# 🧠 Modos de Interação do Copiloto

O Copiloto possui diferentes ***modos de interação*** para atender às diversas necessidades durante o desenvolvimento. Ele pode ser utilizado para ***tirar dúvidas sem alterar o código***, ***editar trechos específicos***, ***planejar mudanças mais complexas*** ou até ***executar tarefas completas de forma mais autônoma***.
A proposta é simples:

> O usuário escolhe o modo que melhor se adapta ao seu objetivo no momento  
> e ganha mais agilidade e controle no processo de desenvolvimento.

## 📚 Study

O modo **Study** é voltado para ***aprendizado ativo***.

>Ele não entrega apenas a resposta pronta.  
>Em vez disso, ajuda a entender o caminho até a solução.

Nesse modo, ele:

- 🧠 Ensina o raciocínio passo a passo  
- ⚖️ Explica conceitos e diferenças importantes  
- ❓ Faz perguntas para estimular o pensamento  
- 📈 Aumenta o nível de dificuldade aos poucos  

Funciona como um ***tutor particular***, ajudando a aprender de verdade — e não apenas resolver o problema.
